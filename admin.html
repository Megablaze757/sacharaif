Show Your Instagram Feed on the Site

The easiest way to automatically display your latest Instagram posts is with an embed widget. These pull your posts directly — no coding beyond a simple paste.

Option 1: LightWidget (Free)

  1. Go to lightwidget.com and sign in with Instagram.
  2. Customise the grid layout, colours and number of posts.
  3. Copy the embed code (it looks like <iframe src="...">).
  4. Paste it into index.html — see "Where to paste" below.

Option 2: Elfsight (Free tier)

  1. Go to elfsight.com and create a free account.
  2. Create an Instagram Feed widget, connect your account and customise.
  3. Copy the <script> and <div> embed code.
  4. Paste into index.html as shown below.

Where to paste the embed code

Open index.html, find the line <!-- Contact -->, and paste this block just above it:

<!-- Instagram Feed --> <section style="padding: 80px 0; background: var(--light);"> <div class="container" style="text-align:center;"> <span class="section-label">Follow Along</span> <h2 class="section-heading" style="margin:0 auto 40px;"> Latest from Instagram </h2> <!-- PASTE YOUR WIDGET EMBED CODE HERE --> </div> </section>

How to Add a New Image to Your Gallery

  1. Prepare your image. Name it with no spaces (use hyphens), e.g. kitchen-island-oak.jpeg. Keep it under 2 MB — compress free at tinypng.com.
  2. Upload the image file to the same folder as your HTML files on your web host.
  3. Use the "Add Images" tab to generate the HTML code block.
  4. Open gallery.html in a text editor (VS Code, Notepad++, or even Notepad).
  5. Find gallery-grid and paste your code just before the closing </div> of that grid.
  6. Save and upload the updated gallery.html. Done!

Updating the Home Page Portfolio

The portfolio section on index.html works the same way — items are inside <div class="portfolio-grid">. Copy an existing portfolio-item block, change the image/title/description, and save.

Image Tips

Format: JPEG for photos. PNG only if you need transparency.

Size: 1200–1600 px on the long edge is ideal. Larger just slows the page.

Naming: Lowercase, hyphens, no special characters. Good: media-wall-led.jpeg. Bad: IMG_1234 (2).JPEG.

Compression: Run images through TinyPNG or Squoosh first — typically cuts 60–80% with no visible quality loss.

Contact Form Setup

The contact form uses Formspree (free, up to 50 submissions/month):

  1. Go to formspree.io and create a free account.
  2. Create a new form and copy the form ID (e.g. xpzvkrwq).
  3. Open index.html, search for YOUR_FORM_ID, and replace it with your ID.
  4. Upload — submissions will go straight to your email.
